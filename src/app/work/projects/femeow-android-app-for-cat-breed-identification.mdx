## Overview

FeMeow is a cutting-edge Android application designed to accurately identify various cat breeds using machine learning. Developed in collaboration with Android developers, machine learning specialists, and cloud computing experts, FeMeow integrates state-of-the-art technology with a user-friendly interface. This app empowers cat owners to better understand their pets by easily identifying different cat breeds through image recognition.

## Key Features

- **Cat Breed Identification**: Utilizes machine learning to identify a wide range of cat breeds with high accuracy.
- **User-Friendly Interface**: Designed for ease of use, allowing users to simply upload a cat photo and receive breed information.
- **Seamless Cloud Integration**: Ensures efficient processing and data handling through cloud-based services.
- **Interactive Features**: Offers users the ability to share and explore cat-related content within the app.

## Technologies Used

- **Kotlin**: For building the Android app.
- **Google Cloud**: For machine learning model integration and cloud storage.

## Challenges and Learnings

One of the main challenges was optimizing the machine learning model to run efficiently on mobile devices without sacrificing accuracy. The app needed to be lightweight yet powerful enough to deliver accurate breed predictions. This required in-depth collaboration between the Android development and machine learning teams, pushing my skills in both mobile optimization and cloud integration.

## Outcome

FeMeow has been well-received by early users, with positive feedback on its accuracy and ease of use. The app not only provides reliable breed identification but also fosters a deeper connection between cat owners and their pets. This project demonstrated the power of combining cutting-edge technology with user-centric design.
